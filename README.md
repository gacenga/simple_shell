simple shell project
shell.h - header file
shell.c - contains main function
display_prompt.c - displays prompt to get input from user
token.c - contains the strtokn function that breaks the given string into a series of tokens using the delimiter passed as a parameter
execute_command.c - contains the execute_command function
my_strdup.c - custom version of strdup()
cd.c - contains function that handles cd inbuilt command
environment.c - contains function that handles setenv and unsetenv
